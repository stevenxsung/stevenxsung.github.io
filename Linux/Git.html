Git

Setup

config

git help config
git config --global user.name "Some Name"
git config --global user.email "your@email.here"
git config --global core.editor vim
git config --list

Getting Started

basic

git init
git clone https://host.name/path repo_name
git clone user@host.name:path/to/repo.git
git pull
git status

commit

git add *
git commit
git push

reset changes

git reset *
git checkout *
git pull

Useful Tips

discard commit

git reset HEAD^
git status
git log

change remote server

git remote -v
git remote set-url origin some_url
git remote add some_head some_url
git remote -v
git pull some_head some_branch

cherry-pick

git checkout some_branch
git branch
git cherry-pick your_commit_id
git push
